The select() function is used to “select” a column from a data.frame type object. The simplest way to use it is giving the name of the data.frame object in first place and then, the name of the column, surrounded by ” “:

DateColumn <- select(Data, "Date")
DateColumn[1:10,]

[1] “2025-07-26” “2025-07-26” “2025-07-26” “2025-07-26” “2025-07-26” “2025-07-26” “2025-07-26” “2025-07-26” “2025-07-26” “2025-07-26”

This results in the column being selected, resulting in the new object containing only that subsetted out column from the original Data object.

In general, the function select() is used with the implementation of pipe operators, not like we used previously:

DateColumn <- Data |> select("Date")
DateColumn[1:10,]

[1] “2025-07-26” “2025-07-26” “2025-07-26” “2025-07-26” “2025-07-26” “2025-07-26” “2025-07-26” “2025-07-26” “2025-07-26” “2025-07-26”

The main advantage of the pipe operators is that we can link different functions. For example, the nrow() function determine the number of rows. We can link the function select() and nrow() with the pipe operators:

::: {.cell}

Data |> select("Date") |> nrow()
